Système de templates

Le système de templates vous permet de séparer entièrement votre code source PHP du code (HTML, CSS, Javascript) de vos sites internet. 

Fichier PHP (index.php)
|
'-- appelle la classe iptTemplate
'-- lit le fichier HTML a afficher (templates/accueil.html)
'-- remplace les valeurs nécessaires dans le fichier
'-- Affiche la page


Voici quelques bases qui vous serviront à profiter au maximum de ce système.

Codes de fusions

Lorsque l'on a un groupe de caractère compris entre les châines "[--" et "--]" (par exemple "[--user_id--]"), normalement il s'agit de mot réservés pour le remplacement directement par votre application PHP avant l'affichage de la page.

Dans la présente application, les codes de fusions suivant sont pris en compte: 
[--content--]....... Est utilisé pour des templates incluant un autre template 
[--baseurl--]....... URL de base sans le nom du fichier appelée (dossier actif)
[--scriptname--].... URL de la page appellée (sans le domaine) sans les variables GET
[--username--]...... Nom d'utilisateur de l'usagé identifié au système 
[--user_prenom--]... Nom de l'usagé identifié au système
[--user_nom--]...... Prénom de l'usagé identifié au système
[--userid--]........ ID de l'usagé identifié au système


Fusion multi-ligne

Il est possible d'utiliser des requêtes ou tableaux pour générer vos pages de contenu. Il vous suffit de placer une balise <\!--row--> au début du bloc de code html et une balise<\!--/row--> à la fin pour que ce dernier soit répéter autant de fois que vous avez de lignes dans votre tableau ou jeu de données.

Chacune des colonnes de votre jeu de données peut être ensuite utilisée via le code de fusion correspondant au nom de la colonne précédé du caractère "{" et suivi du "}".

Exemple:


  
    {nom}
Si vous souhaitez utiliser plusieurs jeux de données, il vous est possible de créer d'autres bloc de fusion en utilisant les "<\!--"" et "-->". Exemples: "<\!--row2-->","<\!--my_row-->","<\!--test de donnees-->".
